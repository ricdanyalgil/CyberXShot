# Segurança

Não publique vulnerabilidades em issues abertas. Envie um aviso privado pelo recurso **Security → Report a vulnerability** deste repositório.

Informe a versão, sistema operacional, passos mínimos para reprodução e impacto observado. Evite anexar capturas com dados pessoais.

Capturas só devem sair do dispositivo após ação explícita do usuário. Mudanças que alterem essa regra precisam de consentimento visível e documentação.
