# Contribuindo

Obrigado por ajudar o CyberXShot.

1. Crie um fork e uma branch curta a partir de `main`.
2. Rode `npm install`, `npm test` e `npm run build`.
3. Mantenha mudanças focadas e inclua testes quando alterar lógica.
4. Abra um pull request explicando comportamento, plataformas testadas e impacto de privacidade.

Ao contribuir, você concorda que seu código será distribuído sob a licença MIT do projeto.
